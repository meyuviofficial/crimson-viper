## Setup

1. Clone and setup:
   ```bash
   git clone https://github.com/meyuviofficial/crimson-viper.git
   cd crimson-viper
   npm run project-setup
   npm install
   ```

2. Run locally:
   ```bash
   npm run dev
   ```
   Visit `http://localhost:1313`

3. Build:
   ```bash
   npm run build
   ```

## Requirements

- Hugo Extended v0.147+
- Node.js v22+
- Go v1.23+

## Deploy

Auto-deployed via GitHub Actions on push to `main` branch.
